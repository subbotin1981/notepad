День 1. Прогресс-бар

<progress> - создает прогресс-бар, который мы можем анимировать через JavaScript.

Загрузка мозга: >Загрузка мозга

Так выглядит тег <progress> без каких либо заданных значений

Синтаксис

<progress value="число" max="число" >

Атрибуты тега <progress>

<max> Максимальное значение прогресса.

<value> Текущее значение прогресса.

<meter> - отображает числовые значения в баре и в зависимости от значения окрашивает строку в определенный цвет.

Используется для вывода значения в некотором известном диапазоне. Используется преимущественно для отображения числовых значений, например, количества результатов поиска, объема жидкости, давления и др.

Атрибуты тега <meter>

<value> Устанавливает значение. Обязательный атрибут.

<min > Задает минимально возможное значение.

<max > Задает максимально возможное значение.

<low > Определяет предел, при достижении которого значение считается низким.

<high > Определяет предел, при достижении которого значение считается высоким.

<optimum > Определяет наилучшее или оптимальное значение.

Пример1 (значение=20):

- оптимальное 54

- оптимальное 14

- оптимальное 89

Пример 2 (нервное состояние):

хорошее - хорошее

волнуется - волнуется

может ударить - может ударить

Если браузер не понимает теги <progress> и <meter> вместо рисунков он отображает текст, который находится в них.

<details> - создает спойлер. Используется для хранения информации, которую можно скрыть или показать по требованию пользователя. По умолчанию содержимое тега не отображается, для изменения статуса применяется атрибут open.

<summary> - создает название для спойлера.

День 2. Псевдоэлементы.

Псевдоэлементы позволяют использовать какие то новые элементы не добавляя в HTML код.

Псевдоэлемент, который используется для вывода желаемого текста после содержимого элемента, к которому он добавляется. Псевдоэлемент :after работает совместно со свойством content .

Для :after характерны следующие особенности.

  • При добавлении :after к блочному элементу, значение свойства display может быть только: block, inline, none, list-item. Все остальные значения будут трактоваться как block.
  • При добавлении :after к встроенному элементу, display ограничен значениями inline и none. Все остальные будут восприниматься как inline.

::after - добавляет псевдоэлемент в конце тега.

::before - добавляет псевдоэлемент в начале тега.

Я сказал ГОРБАТЫЙ

p.testpe:after{ content:"И точка"; color: red;}

Псевдоэлементы ::after и ::before работают при наличии свойства content

Когда не нужен текст для псевдоэлементов можно в контенте указать пустой символ пробел или спецсимвол пробела

::first-letter - задает стили для первого символа

::first-line - задает стили для первой строки

Лучше использовать новые (два двоеточия, старые - одно) псевдоэлементы, тк они работают во всех браузерах.

::selection - позволяет задавать стили к выделенному тексту.
