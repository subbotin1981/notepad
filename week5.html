День 1. Атрибуты тега input.

При помощи дополнительных атрибутов, мы можемболее конкретно настраивать поля наших форм.

Атрибуты тега input

  • required - делает поле обязательным для заполнения
  • placeholder -текст поля который исчезает при фокусе.
  • autocomplete - автозаполнение поля.
  • pattern - задает формат ввода данных для поля.
  • min - задает минримальное значение для поля
  • max - задает максимальное значение
  • multiple - разрешает указать несколько почтовых адресов (через запятую) или файлов для загрузки
  • step - задает шаги актуален для полей числовых и ползунка
  • autofocus - задает автоматический фокус для указанного поля
  • form - привязывает поле к определенной форме.
  • list - используется для привязки к тегу <datalist> что бы указать на список вариантов

День 2. Элементы форм.

<label> - этот тег связывает текст и поле формы тоесть при клике на нексте активируется поле.

Связываться можно двумя способами.
1. Разместить все внутри тега <lable>.
2. При помощи специального атрибта и идентификатора.

<select> - задает список в котором можно выбрать определенные значения. Значения (пункты списка задаются при помощи тега <option>)

<optgroup> - обьединяет пункты списков в отдельные группы

<output> - через этот тег можно вывести результат поля формы или его дополнительно обработать при выводе.

<datalist> - создает список вариантов для заполнения поля. Активируется данные список при фокусе на поле (двойной щелчок) либо при вводе информации.

<textarea> - поле для ввода большого текста. Текст виден в несколько строк.
