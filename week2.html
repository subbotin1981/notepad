День 1. Что такое CSS?

CSS (стили или таблицы стилей) - используется для того что бы управлять дизайном сайта и оформлением каждого элемента страницы.

Споcобы заведения стилей:

Связанные стили
Они находятся внутри отдельного файла стилей, и связываются со всеми страницами сайта при помощи специального тега <link>
Глобальные стили
Они находятся на одной странице с HTML и задаются в теге <head>
Внутренние стили
Задаются внутри тега при помощи специального атрибута style.

Этот текст мы сделали красным при помощи внутренних стилей, задав следующий атрибут: style="color: red;"

Глолбальные стили задаются при помощи тега <style> </style>. Он сообщает браузеру что внутри него идет CSS

Если у вас сайт состоит из множества страниц лучше всего подойдут связанные стили. Глобальные стили используются на одностраничниках.

Важно:

У каждого способа задания стилей есть свой приоритет. Самый высокий приоритет у внутренних стилей, затем идут глобальные и самый низкий у связанных стилей.

B CSS как и в HTML есть свои комментарии и они задаются при помощи: /*текст*/

Синтаксис CSS

Как HTML состоит из тегов так CSS состоит из селекторов. Селекторы указывают к какому элементу/элементам сайта мы хотим применить стили.

В свою очередь каждый селектор состоит из свойств, а каждое свойство состоит из значений. Вот базовый синтаксис:

селектор {свойство: значение;}

Пример использования

body {color: #272727}

У каждого селектора может быть несколько свойств, ограничений по количеству нет, свойства задаются через ; .

селектор {свойство: значение; свойство2: значение; свойство3: значение;...}

Пример использования

body {color: #272727; background: #efefef; }

И у каждого свойства может быть несколько значений. Значения указываются через пробел

селектор {свойство: значение значение2 значение3 значение4 ...;}

Пример использования

body {color: #272727; background: #efefef url(images/bg.png); }

День 2. Селекторы.

Они используются что бы конкретно указать элемент сайта, к которому мы собираемся применить стили.

Для того что бы добраться до любого элемента сайта мы будем использовать различные виды селекторов.

Виды селекторов.

Селекторы тегов - используется что бы конкретно указать теги к которым мы хотим применить стили. В качестве названий используются имена тегов без угловых кавычек!

Пример использования

h2 {border-bottom: 1px dashed #ccc;}
для всех заголовков 2 уровня задали штрих пунктирную границу снизу толщиной в 1 пиксель

Классы (селекторы классов) - нужны что бы применять стили к отдельным элементам сайта выборочно, а не ко всем сразу.

В файле стилей задаются следующим образом:

.имя_класса {стили для него}

в HTML привязываются к тегам при помощи специального атрибута class="имя_класса"

Идентификаторы (селекторы идентификаторы) - нужны что бы применять стили к отдельным элементам сайта выборочно, а не ко всем сразу. Отличаются от классов тем, что они должны быть уникальны и должны быть использоваться ОДИН раз на странице

В файле стилей задаются следующим образом:

#идентификатор {стили для него}

в HTML привязываются к тегам при помощи специального атрибута id = "идентификатор"

Важно:

Имена для идентификаторов и классов могут быть произвольные, однако есть свои ограничения. В именах могут использоваться только латинские буквы а так же знак дефиса и нижнего подчеркивания, так же могут использоваться цифры. И главное имя начинается с латинской буквы.

Важно:

Помните что у идентификаторов приоритет всегда выше, чем у классов

Универсальный селектор - применяет стили сразу для всех элементов сайта

Выглядит он следующим образом:

* {стили}

Селекторы атрибутов - позволяют задать стили тегам у которых есть определенные атрибуты.

Задаются через квадратные кавычки, в которых указываются имя атрибута.

Вот как это выглядит:

[атрибут] {стили}

День 4. Стройка

Два великих тега

<div> - это пустой тег блочного вида, который надо использовавть для задания структуры сайта.

<span> - это пустой тег строчного вида, который надо использовавть для оформления отдельных участков какого то текста.

Блочные элементы - это теги, которые по умолчанию растягиваются на всю ширину родителя. Начинаются с новой строки, и следующий за ними текст тоже переносят на новую строку.

Строчные элементы - это теги, которые не переносят на новую строку.

Важно: блочные элементы автоматом растягиваются на всю ширину родителя, высота у них по умолчанию зависит от содержимого.

Обтекание

Оно задается при помощи свойства float. Если есть обткание то тег которому мы задали обтекание, перестает переносить элементы которые идут после него на новую строку. И эти элементы начинают его обтекать

Важно: если блочному элементу задать обтекание то он перестанет растягиваться на всю ширину родителя

Значение свойства float:

none - оно присваивается по умолчанию

left - оно прижимает элемент к левому краю и все остальные теги которые идут за ним обтекают его справа.

right - оно работает наоборот

Очистка обтекания

За очистку обтекания отвечает свойство clear. Когда мы его задаем, все теги начиная с этого тега с этим свойством перестают обтекать.

Значения свойства clear:

none - очистки нет

left - очистка обтекания тегом со свойством float:left;

right- очистка обтекания тегом со свойством float:right;

both - очистка всех обтеканий.

День 5. Измерители

В стилях есть свойства которые влияют на размер блоков.

Длина и высота

Свойство height - указывает на высоту блока.

Свойство width - указывает ширину блока.

Важно: помните что размеры мы можем указывать только для блочных типов элементов. Для строчных они работать не будут.

Значения и единицы измерения:

auto - задает значение высоты по умолчанию. Ширина блочных элементов зависит от родитля.

% - указывают размер блока в процентах. Размер вычисляется относительно родителя.

px/in/pt - количественные, точные значения.

Важно: если картинке прописать только одну сторону, а для второй указать auto, то она автоматически выровняется согласно пропорциям.

Границы

В первую очередь помните что любая граница созданная при помощи свойства border влияет на размер в звисимости от своей толщины.

border - указывает границу , по умолчанию она отсутсвует у многих элементов (тегов) сайта.

Синтаксис

border: толщина тип цвет;

Толщину можно указывать любую в количественных единицах измерения.

Типы границ:

dotted dashed solid

далее толщина 5px

outset inset groove double ridge

Ширина этого блока указана в 300px, однако она на самом деле составляет 320px, так как у этого блока задана граница толщиной в 10px.

Как указывать границы отдельно:

border-top - граница сверху

border-right- граница справа

border-bottom- граница снизу

border-left- граница слева

У этого блока разные границы так как мы использовали разные стили.

Поля и отступы.

Поля указывают внутренние отступы, а отступы - внешние.

padding - это свойство задает внутренние отступы или поля. Влияет на размер всех элементов, которым оно задано.

Поля можно указывать отдельно:

padding-top - поля сверху

padding-right- поля справа

padding-bottom- поля снизу

padding-left- поля слева

Можно указывать поля разными способами:

padding: верх право низ лево;

padding: верх право_лево низ

padding: верх_низ право_лево

Ширина это блока 300 пискселей, но на самом деле у него ширина 342 пикселя - 40 пикселей за счет паддингов, и еще 2 за счет границ

margin - задает внешние отступы. Они рабоатают точно так же как и поля, за тем исключением что они не влияют на размер блока.

Важно:для строчных элементов внешние отступы не работают по вертикали, только справа и слева.
