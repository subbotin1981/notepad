День 1. Что такое CSS?

CSS (стили или таблицы стилей) - используется для того что бы управлять дизайном сайта и оформлением каждого элемента страницы.

Споcобы заведения стилей:

Связанные стили
Они находятся внутри отдельного файла стилей, и связываются со всеми страницами сайта при помощи специального тега <link>
Глобальные стили
Они находятся на одной странице с HTML и задаются в теге <head>
Внутренние стили
Задаются внутри тега при помощи специального атрибута style.

Этот текст мы сделали красным при помощи внутренних стилей, задав следующий атрибут: style="color: red;"

Глолбальные стили задаются при помощи тега <style> </style>. Он сообщает браузеру что внутри него идет CSS

Если у вас сайт состоит из множества страниц лучше всего подойдут связанные стили. Глобальные стили используются на одностраничниках.

Важно:

У каждого способа задания стилей есть свой приоритет. Самый высокий приоритет у внутренних стилей, затем идут глобальные и самый низкий у связанных стилей.

B CSS как и в HTML есть свои комментарии и они задаются при помощи: /*текст*/

Синтаксис CSS

Как HTML состоит из тегов так CSS состоит из селекторов. Селекторы указывают к какому элементу/элементам сайта мы хотим применить стили.

В свою очередь каждый селектор состоит из свойств, а каждое свойство состоит из значений. Вот базовый синтаксис:

селектор {свойство: значение;}

Пример использования

body {color: #272727}

У каждого селектора может быть несколько свойств, ограничений по количеству нет, свойства задаются через ; .

селектор {свойство: значение; свойство2: значение; свойство3: значение;...}

Пример использования

body {color: #272727; background: #efefef; }

И у каждого свойства может быть несколько значений. Значения указываются через пробел

селектор {свойство: значение значение2 значение3 значение4 ...;}

Пример использования

body {color: #272727; background: #efefef url(images/bg.png); }

День 2. Селекторы.

Они используются что бы конкретно указать элемент сайта, к которому мы собираемся применить стили.

Для того что бы добраться до любого элемента сайта мы будем использовать различные виды селекторов.

Виды селекторов.

Селекторы тегов - используется что бы конкретно указать теги к которым мы хотим применить стили. В качестве названий используются имена тегов без угловых кавычек!

Пример использования

h2 {border-bottom: 1px dashed #ccc;}
для всех заголовков 2 уровня задали штрих пунктирную границу снизу толщиной в 1 пиксель

Классы (селекторы классов) - нужны что бы применять стили к отдельным элементам сайта выборочно, а не ко всем сразу.

В файле стилей задаются следующим образом:

.имя_класса {стили для него}

в HTML привязываются к тегам при помощи специального атрибута class="имя_класса"

Идентификаторы (селекторы идентификаторы) - нужны что бы применять стили к отдельным элементам сайта выборочно, а не ко всем сразу. Отличаются от классов тем, что они должны быть уникальны и должны быть использоваться ОДИН раз на странице

В файле стилей задаются следующим образом:

#идентификатор {стили для него}

в HTML привязываются к тегам при помощи специального атрибута id = "идентификатор"

Важно:

Имена для идентификаторов и классов могут быть произвольные, однако есть свои ограничения. В именах могут использоваться только латинские буквы а так же знак дефиса и нижнего подчеркивания, так же могут использоваться цифры. И главное имя начинается с латинской буквы.

Важно:

Помните что у идентификаторов приоритет всегда выше, чем у классов

Универсальный селектор - применяет стили сразу для всех элементов сайта

Выглядит он следующим образом:

* {стили}

Селекторы атрибутов - позволяют задать стили тегам у которых есть определенные атрибуты.

Задаются через квадратные кавычки, в которых указываются имя атрибута.

Вот как это выглядит:

[атрибут] {стили}
